# Курсовая работа по классическому машинному обучению

Репозиторий содержит курсовую работу студента 1 курса магистратуры НИЯУ МИФИ (М245583).

**Краткое описание задачи:** необходимо построить прогноз, позволяющий подобрать наиболее эффективное сочетание параметров для создания лекарственных препаратов.

**Структура проекта**:

**Файлы(.ipynb):**
* `EDA` — разведочный анализ данных и предобработка
* `Regression_IC50` — регрессионные модели для прогнозирования IC50
* `Regression_CC50` — регрессионные модели для прогнозирования CC50
* `Regression_SI` — регрессионные модели для прогнозирования SI
* `Classification_IC50` — классификация: превышает ли значение IC50 медианное значение
* `Classification_CC50` — классификация: превышает ли значение CC50 медианное значение
* `Classification_SI` — классификация: превышает ли значение SI медианное значение
* `Classification_SI_8` — классификация: превышает ли индекс селективности (SI) порог > 8

**Аналитический отчет:** `Отчет.pdf` — выводы о проделанной работе, сравнение моделей и итоговое обоснование выбора лучших алгоритмов.
